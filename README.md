# Mustag Culture — Portfolio Site

A single-page portfolio for Adeolu / Mustag Culture: custom tailoring, ready-to-wear, and fashion consultation.

## Files
- `index.html` — page content
- `style.css` — all styling
- `script.js` — scroll progress bar + mobile menu
- `images/` — the six portfolio photos

## Host it on GitHub Pages
1. Create a new repository on GitHub (e.g. `mustang-culture`).
2. Upload all files in this folder, keeping the `images/` folder intact.
3. Go to the repo's **Settings → Pages**.
4. Under "Build and deployment", set Source to **Deploy from a branch**, branch `main`, folder `/ (root)`.
5. Save. GitHub will give you a live URL like `https://yourusername.github.io/mustang-culture/` within a minute or two.

## Editing later
- Swap or add photos in `images/`, then update the matching `src` in `index.html`.
- Update the email, WhatsApp link, and Instagram handle in the Contact section of `index.html`.
- Colours and fonts are defined as CSS variables at the top of `style.css` under `:root`.
